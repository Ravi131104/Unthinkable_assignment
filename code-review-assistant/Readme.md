# Code Review Assistant

An automated code review tool that uses the Gemini 2.0 Flash LLM to analyze source code for structure, readability, and best practices.

## Project Overview
This tool helps developers improve their code by providing instant AI-generated feedback. It meets the following project requirements:
- **Backend:** Fast & Asynchronous API using FastAPI.
- **AI Engine:** Powered by Google Gemini 2.0 Flash for deep code analysis.
- **Frontend:** Modern, Responsive Dark Mode Dashboard with Markdown rendering.
- **Real-time Feedback:** Loading states and formatted reports.
